== Description ==

Information For Help gives you the collection of the useful information you should share when you ask help for your website.

No settings for this plugin.

To get the needed information follow these steps:
- click on action link “Get information for help” or go to the admin page “Information for Help”
- click on “Copy for Support”
- paste the copied information in the message you will send to people who help you


Using the actual version of Information For Help you can have a collection of the following information:

- Home URL

- Wordpress Version

- Wordpress Multisite (active/not active)

- Wordpress Debug Mode (active/not active)

- Host

- PHP Version

- MySQL Version

- Database size

- Must Use Plugins

- Active Plugins

- Current Theme

- Theme Version

- Child Theme

- User Agent


== Installation ==

1. Upload the entire `information-for-help` folder to the `/wp-content/plugins/` directory or install it using the usual installation button in the Plugins administration page.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You will see the Information for Help menu item in your admin menu
4. All done. Good job!
